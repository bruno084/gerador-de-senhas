#  Gerador de Senhas em Python

Este é um dos meus primeiros projetos na minha jornada de programação! É um script simples que gera senhas aleatórias seguras com base na quantidade e no tamanho que o usuário escolher.

##  Como funciona?
O programa junta letras (maiúsculas e minúsculas) e números, e depois sorteia os caracteres para criar senhas únicas.

##  Tecnologias Utilizadas
- **Python 3**
- Biblioteca padrão `random` (usada para sortear os caracteres)
